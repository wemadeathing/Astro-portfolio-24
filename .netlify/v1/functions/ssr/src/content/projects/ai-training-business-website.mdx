## Project Overview
My AI training business where I help people learn practical AI skills that actually work in their daily lives. Built the entire platform using Cursor and Claude Code to practice what I preach - using AI tools to build efficiently while maintaining quality. The focus is on practical application rather than theoretical knowledge, reflecting my own learning-by-building approach to mastering new technologies.

## Key Features

### Education Platform
A modern training platform that reflects the practical, hands-on approach I teach:

<ul class="list-none space-y-3 [&>li]:before:content-['•'] [&>li]:before:text-purple-600 [&>li]:before:mr-2 [&>li]:flex">
  <li>Course Management - Built with MDX for easy course content updates and interactive examples</li>
  <li>Student Experience - Clean, focused interface that prioritizes learning over flashy features</li>
  <li>Practical Focus - Content emphasizes real-world application rather than theoretical AI concepts</li>
  <li>Modern Architecture - Astro with React islands for optimal performance and developer experience</li>
  <li>Email Integration - Automated student communications and course updates using Resend</li>
  <li>Database Management - Neon DB for student progress tracking and course analytics</li>
</ul>
